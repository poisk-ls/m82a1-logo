# m82a1-logo
Convert Your Pictures/Logos into ASCII Art

# Disctiption:
<big>Using this Tool, You can convert your Photos and Logos into ASCII Art..
Which can be printed in Terminals</big>

- pkg install python

- pit clone https://github.com/poisk-ls/m82a1-logo
